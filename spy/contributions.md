# 🔎 Open Source Search Engine

## 📖 Description

This project is an open source search engine designed to crawl, render, and index websites efficiently. It supports both static and JavaScript-heavy pages using a hybrid architecture powered by Python and Node.js. The system is built to mimic basic behavior of modern search engines by combining crawling, rendering, parsing, and storage.

# 🤝 Contributing Guide

Thank you for your interest in contributing! 🚀
We welcome all kinds of contributions.

---

## 📌 How to Contribute

### 1. Fork the Repository

Click on the **Fork** button and create your own copy.

---

### 2. Clone the Repo

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

---

### 4. Make Changes

* Write clean and readable code
* Follow project structure
* Add comments if needed

---

### 5. Commit Changes

```bash
git add .
git commit -m "Added: short description"
```

---

### 6. Push to GitHub

```bash
git push origin feature/your-feature-name
```

---

### 7. Create Pull Request

Go to GitHub and click **New Pull Request**

---

## 🛠️ Contribution Types

* 🐛 Bug fixes
